FlowForge Automator User Guide

Welcome to FlowForge Automator - The ultimate tool for automating Video and Image creation on the Flow platform.

üöÄ 1. INSTALLATION

The tool operates as a Chrome Extension.

  1. Download and extract the tool's .zip file into a folder (e.g., FlowForge_v3).
  2. Open your Google Chrome browser.
  3. Go to the Extension management page:
    • Enter chrome://extensions/ in the address bar and press Enter.
    • Or go to Menu (3 dots) > Extensions > Manage Extensions.
  4. Enable Developer mode (Top right corner).
  5. Click the Load unpacked button (Top left corner).
  6. Select the folder you just extracted (FlowForge_v3 or dist_safe).
  7. The FlowForge icon will appear in your Chrome toolbar.

üîë 2. LICENSE TIERS

The tool offers 4 account levels with different benefits:

1. Free Mode (Default)

  • Default upon installation.
  • Daily Quota: 5 runs per day.
  • Max Prompts: Up to 5 prompts per run.
  • Batch Mode: Allowed, but Batch Size is limited to 2.
  • Devices: 1 device.

2. Trial Tier (Time-Limited)

  • For users wanting to experience full features for a short period (e.g., 7 days).
  • Daily Quota: 20 runs per day.
  • Max Prompts: Up to 20 prompts per run.
  • Batch Mode: Batch Size based on server config (usually 2-3).

3. Standard Tier (Lifetime)

  • A lifetime paid plan.
  • Daily Quota: 20 runs per day.
  • Max Prompts: Up to 20 prompts per run.
  • Batch Mode: Batch Size based on server config (usually 2-3).

4. Premium Tier (Unlimited)

  • The highest tier with full power.
  • Daily Quota: Unlimited (9999 runs).
  • Max Prompts: Unlimited prompts (up to 999).
  • Batch Mode: Max Batch Size of 5 (Highest speed on Flow).
  • Devices: Supports up to 3 devices simultaneously.
  • Custom Sequence: Enables the "Enable Custom Sequence" feature.

How to Activate:

  1. Click the FlowForge extension icon to open the Sidepanel.
  2. Click the "Upgrade" button or the key icon üîë.
  3. Enter the License Key you purchased.
  4. Click the activation button.
  5. If successful, the corresponding badge (TRIAL, STANDARD, PREMIUM) will appear.

üé¨ 3. FEATURE: VIDEO TAB

Used to create videos from text/image descriptions.

A. Main Interface:

  • Prompt List: Enter a list of prompts describing your video. Each line is a separate prompt.

B. Operation Modes:

  1. Sequential Mode (Default):
    • Processes each prompt line by line from top to bottom.
    • Quota: Deducts 1 credit per execution.
    • Best for controlling each individual video generation.
  2. Batch Mode (High Speed) üöÄ:
    • Turn on the "Batch Mode" toggle.
    • Creates up to 5 slots simultaneously.
    • The tool fills slots as they finish to ensure no idle time.
    • Quota: Deducts 2 credits per batch run (for the entire list of prompts).

C. Image Selection:

  • Choose Images: Click to select image files from your computer (Multiple selection supported).
  • Remove Images: Click on an image thumbnail to remove it.

D. Image Pairing Modes:

  1. None (Text-to-Video only): No images used.
  2. 1-to-1 (Frames-to-Video): Each prompt uses 1 image as the Start Frame.
  3. Static (Frames-to-Video): Uses 1-2 fixed images for all prompts.
  4. Start-End (Frames-to-Video): Uses pairs of 2 images (Start & End) for each prompt.
  5. Ingredient 1-to-1: Upload up to 3 reference images for each prompt.
  6. Ingredient Static: Upload up to 3 reference images to be used for all prompts.

üé® 4. FEATURE: CREATE IMAGE TAB

A. Main Interface:

Used to create images from text. Automatically switches Flow interface to "Create Image" mode.

  • Prompts: Enter descriptions for your images.
  • Batch Mode: Processes multiple prompts simultaneously.
  • Upscale: Automatically supports upscaling to 2K/4K.

B. Image-to-Image Mode:

  • Select Images: Click to upload up to 10 reference images.
  • Pairing Modes:
    1. None: No images used.
    2. 1-to-1: Each prompt uses a specific number of images.
    3. Static: Uses all uploaded images as references for every prompt.

C. Quota Image:

  • Sequential: Deducts 1 credit per run.
  • Batch: Deducts 1 credit per run.

Common Issues:

  • Mode Mismatch: Tool tab doesn't match Flow website mode. -> Fix: Switch web mode.
  • Limit Exceeded: Prompt count or Batch size exceeds tier limit. -> Fix: Upgrade to Standard/Premium.
  • Daily Quota Exhausted: Runs used up. -> Fix: Wait 24h or upgrade.
