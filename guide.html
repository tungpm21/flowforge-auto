‚öôÔ∏è III. USAGE GUIDE

Note: You must be on the Flow working interface.

Example: To create Text to Video, you must have Video Tab open on the tool - AND VICE VERSA. Same Video creation but different modes (e.g., text to video vs frame to video) will not work.

If not matched, an error message will appear -> Just adjust and it will work.

1. General Overview

  • There are 2 interfaces (2 tabs) for video and image creation: Video Tab and Image Tab.
  • There are 2 Auto run features:
    • Sequential: Runs prompts one by one.
    • Batch: Runs multiple prompts simultaneously.

2. Configuration Settings & Features

a. File Name and Download Quality Configuration

Output files will be saved in the FlowVideos folder. File names follow prompt order and prefix.

b. Advanced Configuration

You can customize settings as needed (Delay, Max Retries, etc.).

c. Image Upload Configuration

Used for Image to Video or Image to Image mode.

d. Prompt Input Configuration

  • Each prompt separated by one line - with prompt counter.
  • Buttons available: Save prompt, Load prompt, Upload .txt file, Export .txt file, Clear prompts.
  • Custom sequence mode: Can repeat prompts (Lifetime only).

e. Other Run Configurations

  • Normal Sequential mode: Downloads each prompt after completion.
  • Normal Batch mode: Creates all prompts at once, then downloads later.
  • Create Only: Only creates - no download (Lifetime only).
  • Download All: Only downloads - same logic as Batch mode (Lifetime only).

f. Other Features

  • Progress bar and Preview of completed tasks + Timer.
  • Logs section for monitoring & progress reports.
  • Failed Prompts indicator - shows which prompts failed.

3. Effective Workflow Steps

NOTE:
  • Set and match the correct mode and configuration.
  • Recommend using Batch mode for optimal speed.
  • Check prompts for potential policy violations.
  • Fastest and smoothest at 720P (video) and 1K (Image) quality - no upscale button click needed, downloads directly upon completion.

  1. Step 2: Set configuration on Tool.

Feature Mode Cost Note

Max Batch Size Limit:

  • Free: 2 prompts at once.
  • Trial/Standard: 3 prompts at once.
  • Unlimited - Lifetime: 5 prompts at once.

Common Issues:

  • Mode Mismatch: If you're on Video Tab in the tool, but the Flow website is on "Create Image" page -> Tool will show red warning.
    Fix: Switch the Flow web mode to match the tool's selected mode.
  • Limit Exceeded: Prompt count or Batch Size exceeds your tier's limit.
    Fix: Remove some prompts or upgrade to Standard/Premium.
  • Quota Exhausted: Daily runs used up.
    Fix: Wait 24 hours or upgrade.
